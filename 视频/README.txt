需要安装moviepy库（如果你还没有安装的话）

然后，使用这个Python脚本来切割视频。

脚本中的chunk_duration_seconds参数设置为100秒，因此每个切割出的片段都将至少持续100秒（由于视频总时长为48分06秒也就是2886秒，为了保证每个视频片段都为20秒以上，我选择了将视频切割成每个片段100秒，这样最后一个视频也不至于低于20秒）。

此外，请确保将video_path替换为你的视频文件路径，并根据需要调整output_folder的值以指定输出文件夹的路径。

此脚本将视频切割成多个片段，并将它们保存到指定的输出文件夹中。每个片段的文件名格式为chunk_XXX.mp4，其中XXX是片段的编号（从001开始）。
